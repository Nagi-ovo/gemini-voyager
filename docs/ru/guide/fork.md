# Ветвление разговора (Экспериментально)

Мышление не должно быть улицей с односторонним движением. При сложных исследованиях часто возникает необходимость вернуться к ключевой точке и опробовать различные варианты.

С помощью функции **Ветвления**, Voyager позволяет вам расширять ход мыслей и исследовать параллельные вселенные вашего чата.

## Как это работает

> **⚠️ Примечание**: Это экспериментальная функция. Сперва вам нужно включить ее, нажав на значок расширения на панели инструментов браузера, чтобы открыть всплывающее окно настроек, и активировав переключатель **«Включить ветвление разговора»**.

Для того чтобы пойти по иному пути, просто наведите курсор на ваш вопрос и нажмите кнопку **Ветвление**:

![Ветвление](/assets/branching.png)

Voyager скопирует весь контекст с самого начала до этой точки и покажет окно подтверждения. Выберите вариант по длине контекста:

- **Скачать MD** (рекомендуется для большинства разговоров): у поля ввода Gemini есть ограничение по длине, поэтому длинный контекст может не поместиться при прямой вставке. Voyager скачает Markdown-файл с контекстом и откроет новый разговор; перетащите файл `.md` в Gemini до окончания 2-минутного обратного отсчета в правом нижнем углу. Поле ввода будет заранее заполнено короткой заметкой о том, что вложение является контекстом предыдущего разговора, и местом для нового запроса.
- **Fork** (лучше для коротких разговоров): если контекст короткий, Voyager откроет новый разговор и заполнит поле ввода напрямую; отправьте его, чтобы создать ветку.

После отправки Voyager записывает только связь ветки. Он не удаляет и не переписывает исходный разговор.

В этой новой ветке вы можете свободно изменять свой вопрос и исследовать другие направления, не беспокоясь о том, что разрушите оригинальную историю чата. Дайте волю своей креативности и любопытству!
