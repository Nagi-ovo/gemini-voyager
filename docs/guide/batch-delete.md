# 批量删除

在 Gemini 和 AI Studio 一次性删除多个对话，告别逐个删除的繁琐操作。

## 功能介绍

- **多选模式**：长按任意对话进入多选模式，可勾选多个要删除的对话。
- **一键清理**：选中后点击删除按钮，批量删除所有选中的对话。
- **进度反馈**：删除过程中显示实时进度，让你了解当前状态。
- **安全确认**：删除前会弹出确认对话框，防止误操作。

## 如何使用

### Gemini

1. 在侧边栏的对话列表中，**长按**任意一个对话项。
2. 进入多选模式后，对话项左侧会出现复选框。
3. 勾选你想要删除的对话（一次最多可选 50 个）。
4. 点击出现的 **删除按钮**。
5. 在**文件夹列表上方**出现的红色确认区域中点击“确定”，即可开始批量删除。

### AI Studio

1. 打开 AI Studio 的 **Library** 页面。
2. **长按**任意 prompt 行进入多选模式。
3. 选中要删除的 prompt（一次最多可选 50 个）。
4. 点击浮动工具条里的 **删除按钮**，再确认浏览器弹窗。

::: tip 提示
在 Gemini 中，删除确认面板会直接覆盖在文件夹区域上方，以免遮挡对话列表。批量删除操作无法撤销，请谨慎操作。
:::
