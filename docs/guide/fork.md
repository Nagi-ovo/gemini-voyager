# 对话分支 (实验性)

思维不应是一条单行道。在复杂的探索中，我们常常需要回到某个关键节点，尝试不同的可能性。

Voyager 带来的 **对话分支** 功能，让你能够轻松发散思维，探索对话的平行宇宙。

## 功能介绍

> **⚠️ 提示**：该功能目前处于实验阶段。你需要先点击浏览器扩展图标打开设置弹窗，并开启 **启用对话分支** 开关。

在任何你想要发散思绪的时刻，只需将鼠标悬停在你的提问上，点击 **对话分支** 按钮：

![对话分支](/assets/branching.png)

Voyager 会截取从对话开头到该节点的所有上下文，并弹出确认框。建议按上下文长度选择：

- **下载 MD**（推荐用于大多数对话）：Gemini 输入栏有长度限制，较长上下文直接填入可能放不完整。Voyager 会下载一份 Markdown 上下文文件并打开新对话；请在右下角 2 分钟倒计时内把 `.md` 文件拖入 Gemini。输入框会预填一句话说明附件是上一段对话的上下文，并留下 **新问题：** 供你继续输入。
- **分叉**（适合较短对话）：上下文较短时，Voyager 会打开新对话并把上下文直接填入输入框；发送后即可创建分支。

发送后，Voyager 只记录分支关系，不会删除或改写原始对话。

你可以在这个新分支中尽情修改提问，尝试不同的方向，而不必担心破坏原有的对话历史。尽情释放你的创造力与好奇心吧！
