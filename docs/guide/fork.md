# 对话分支 (实验性)

思维不应是一条单行道。在复杂的探索中，我们常常需要回到某个关键节点，尝试不同的可能性。

Voyager 带来的 **对话分支** 功能，让你能够轻松发散思维，探索对话的平行宇宙。

## 功能介绍

> **⚠️ 提示**：该功能目前处于实验阶段。你需要先点击浏览器扩展图标打开设置弹窗，并开启 **启用对话分支** 开关。

在任何你想要发散思绪的时刻，只需将鼠标悬停在你的提问上，点击 **对话分支** 按钮：

![对话分支](/assets/branching.png)

Voyager 会截取从对话开头到该节点的所有上下文，并弹出确认框：

- 点击 **分叉**：Voyager 会为你打开一段新对话，并自动把上下文填入输入框。确认内容后发送，即可创建这个分支。
- 点击 **下载 MD**：Voyager 会下载一份 Markdown 上下文文件，并打开一段新对话。请在右下角倒计时结束前（2 分钟内）把 `.md` 文件拖入 Gemini 输入区；输入框会预先填入一段模板，说明附件是上一段对话的上下文，并留下 **新问题：** 供你继续输入。发送后，新对话会被记录为这个节点的分支。

由于 Gemini 的附件拖入流程无法稳定由扩展自动完成，MD 模式需要你手动拖入文件。倒计时提示会显示剩余时间，过期后这次分支关联会失效。

Voyager 只记录分支关系，不会删除或改写原始对话。

你可以在这个新分支中尽情修改提问，尝试不同的方向，而不必担心破坏原有的对话历史。尽情释放你的创造力与好奇心吧！
