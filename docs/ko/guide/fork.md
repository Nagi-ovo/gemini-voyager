# 대화 분기 (실험적)

생각은 일방통행이 되어서는 안 됩니다. 복잡한 탐색에서는 종종 중요한 노드로 돌아가 다른 가능성을 시도해야 할 때가 있습니다.

Voyager의 **대화 분기** 기능을 사용하면 생각을 발산하고 대화의 평행 우주를 자유롭게 탐색할 수 있습니다.

## 기능 소개

> **⚠️ 참고**: 이 기능은 현재 실험적 기능입니다. 먼저 브라우저 툴바의 확장 프로그램 아이콘을 클릭하여 설정 팝업을 열고 **"대화 분기 활성화"** 스위치를 켜야 합니다.

다른 방향으로 나아가고 싶을 때는 언제든지 질문 위에 마우스를 올리고 **분기** 버튼을 클릭하세요:

![대화 분기](/assets/branching.png)

Voyager는 처음부터 해당 지점까지의 모든 컨텍스트를 캡처한 뒤 확인 창을 표시합니다. 컨텍스트 길이에 맞게 선택하세요:

- **MD 다운로드**(대부분의 대화에 권장): Gemini 입력창에는 길이 제한이 있어 긴 컨텍스트를 직접 넣으면 다 들어가지 않을 수 있습니다. Voyager가 컨텍스트가 담긴 Markdown 파일을 다운로드하고 새 대화를 엽니다. 오른쪽 아래 2분 카운트다운이 끝나기 전에 `.md` 파일을 Gemini로 드래그하세요. 입력창에는 첨부 파일이 이전 대화의 컨텍스트임을 알리는 짧은 문장과 새 요청을 쓸 공간이 미리 입력됩니다.
- **Fork**(짧은 대화에 적합): 컨텍스트가 짧을 때 Voyager가 새 대화를 열고 입력창에 직접 채웁니다. 전송하면 분기가 생성됩니다.

전송 후 Voyager는 분기 관계만 기록합니다. 원래 대화를 삭제하거나 다시 쓰지 않습니다.

이 새로운 분기에서는 원래의 대화 기록을 훼손할 걱정 없이 질문을 자유롭게 수정하고 다양한 방향을 탐색할 수 있습니다. 당신의 창의력과 호기심을 마음껏 펼치세요!
