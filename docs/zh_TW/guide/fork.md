# 對話分支 (實驗性)

思維不應是一條單行道。在複雜的探索中，我們常常需要回到某個關鍵節點，嘗試不同的可能性。

Voyager 帶來的 **對話分支** 功能，讓你能夠輕鬆發散思維，探索對話的平行宇宙。

## 功能介紹

> **⚠️ 提示**：該功能目前處於實驗階段。你需要先點擊瀏覽器擴充功能圖示打開設定彈窗，並開啟 **啟用對話分支** 開關。

在任何你想要發散思緒的時刻，只需將滑鼠懸停在你的提問上，點擊 **對話分支** 按鈕：

![對話分支](/assets/branching.png)

Voyager 會截取從對話開頭到該節點的全部上下文，並彈出確認框：

- 點擊 **Fork**：Voyager 會為你開啟一段新對話，並自動把上下文填入輸入框。確認內容後送出，即可建立這個分支。
- 點擊 **下載 MD**：Voyager 會下載一份 Markdown 上下文檔案，並開啟一段新對話。請在右下角倒數結束前（2 分鐘內）把 `.md` 檔案拖入 Gemini 輸入區；輸入框會預先填入一段模板，說明附件是上一段對話的上下文，並留下 **新問題：** 供你繼續輸入。送出後，新對話會被記錄為這個節點的分支。

由於 Gemini 的附件拖入流程無法穩定由擴充功能自動完成，MD 模式需要你手動拖入檔案。倒數提示會顯示剩餘時間，過期後這次分支關聯會失效。

Voyager 只記錄分支關係，不會刪除或改寫原始對話。

你可以在這個新分支中盡情修改提問，嘗試不同的方向，而不必擔心破壞原有的對話歷史。盡情釋放你的創造力與好奇心吧！
